## Como Iniciar

### Instalación

1. Clonación del Repositorio
   ```sh
   git clone https://github.com/juanccs10/FinaktivaTest
   ```
2. Inicio de la API 
   ```sh
   cd FinaktivaApi/FinaktivaApi
   dotnet run
   ```
3. Inicio del Cliente 
   ```sh
   cd FinaktivaClients
   npm install
   ionic serve
   ```
